The program used CMD (Windows command prompt) to interact with users as UI

There is only one owner account: username is "admin" and password is "admin"

Steps to run the program:
1. download the program from GitHub.
2. run all sql files on your local pgAdmin to create database. All sql files are stored in SQL repository.
3. find the executeable jar file in the repository (out->artifact->FinalProject.jar)
4. copy the path of the folder which contains jar file.
5. open Windowns command prompt
6. use command "cd "+ the folder path on step 4.
7. use command "java -jar FinalProject.jar" to execute the program

(Note: the database connection information may be different, which is the port number of the database, the database name, the username of the database)
